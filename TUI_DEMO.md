# TUI 界面演示

## 启动方式

```bash
./target/release/offline-analysis-agent --gui tui
```

## 界面布局

```
┌─────────────────────────────────────────────────────────────────┐
│ 离线分析 AI Agent v0.1.0                                        │
├─────────────────────────────────────────────────────────────────┤
│ [任务管理] │ 元数据 │ 创建向导 │ 数据质量 │ 设置                │
├─────────────────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ 任务列表                                                     │ │
│ │                                                              │ │
│ │ sync_user_daily [增量同步] - 每日用户数据同步               │ │
│ │ 状态：Success                                               │ │
│ │                                                              │ │
│ │ agg_order_hourly [单表聚合] - 每小时订单聚合                │ │
│ │ 状态：Running                                               │ │
│ └─────────────────────────────────────────────────────────────┘ │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│ q:退出 | ←/→:切换 | Enter:操作                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 快捷键

| 按键 | 功能 |
|------|------|
| `q` | 退出应用 |
| `←` / `→` | 切换标签页 |
| `Enter` | 执行操作（开发中） |

## 标签页说明

1. **任务管理**: Azkaban 任务列表，支持 7 种任务模板
2. **元数据**: Hive Metastore 数据库和表浏览
3. **创建向导**: 7 步创建 Azkaban 任务（开发中）
4. **数据质量**: 8 类数据质量检查（开发中）
5. **设置**: 系统配置（开发中）

## 技术栈

- **ratatui 0.24**: Rust TUI 框架
- **crossterm 0.27**: 跨平台终端控制
- 无需图形环境，SSH 直连可用
